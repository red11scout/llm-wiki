## Description

Multi-agent cognitive analysis platform featuring a Document Agent, parallel Strategy and Financial agents, and an Orchestrator agent. Uses Drizzle ORM with .returning() on inserts for verification, BlueAlly brand colors, and Framer Motion animations. An earlier-generation analysis tool that pioneered the multi-agent architecture pattern.

## Ecosystem role

Earlier generation of the multi-agent analysis platform. Its data model and agent architecture were combined with ResearchApp data in the ai-catalyst project, which was later superseded by aiworkflow. Remains deployed on Replit as a reference implementation.
