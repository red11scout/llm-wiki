shadcn/ui is an open-source collection of accessible, composable React UI components built on top of Radix UI primitives and styled with [[tailwindcss]]. Unlike a traditional npm package, components are copied directly into the project codebase, giving full ownership and customization control.

## Usage in BlueAlly ecosystem

[[aiworkflow]] uses 20 shadcn/ui components across its 12-page interface, making it the primary UI component library for that app.
