The [[blueally-pipeline]] has a clear generational lineage: [[cognition-two]] (earliest multi-agent prototype, Feb 2026 on Replit) → [[ai-catalyst]] and [[cognoresearcher]] (second generation, archived Feb 2026) → [[researchapp]] + [[workflow]] (third generation, Feb–Mar 2026) → current [[researchapp-v2]] + [[aiworkflow]] + [[ai-architecture-studio]] stack. Each generation added more structure to the [[multi-agent-pipeline]] pattern. [[smart-report-ai]] co-occupies step 1 alongside researchapp, suggesting parallel experimentation rather than clean sequential replacement.
