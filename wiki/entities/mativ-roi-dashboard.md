Mativ ROI Dashboard is a deployed interactive ROI analysis dashboard built as an external-client deliverable for [[mativ-holdings]]. It presents return-on-investment analysis and business impact metrics through animated data visualizations powered by [[framer-motion]]. All content is statically defined — the tool carries no database layer.

## Technical stack

- **Framework**: [[nextjs]] with [[react]]
- **Styling**: [[tailwindcss]]
- **Animation**: [[framer-motion]] for animated data visualizations
- **Database**: none — static content
- **Deployment**: [[vercel]] at mativ-roi-dashboard.vercel.app
- **Last commit**: 2026-03-24

## Ecosystem role

A standalone external-client deliverable, not connected to the core [[blueally-pipeline]]. It demonstrates BlueAlly's capability to deliver custom analytics and [[roi-dashboard]] tooling for enterprise clients outside the standard pipeline flow. Sits alongside [[ga-staff-scheduler]] as an example of bespoke client-facing tool delivery.
