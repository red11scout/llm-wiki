## Summary

[[ai-executive-briefing]] is a deployed BlueAlly web application that serves as an interactive, animated executive briefing on AI strategy. It was built as a web-native replacement for static PowerPoint slides, using [[framer-motion]] for animations and [[zustand]] for slide navigation and presenter controls.

## Key facts

- **GitHub**: https://github.com/red11scout/ai-executive-briefing

- **Stack**: Next.js, React, Tailwind CSS, [[framer-motion]], [[zustand]]
- **Database**: none
- **Status**: deployed
- **Last commit**: 2026-03-28

## Ecosystem role

Standalone presentation tool for executive audiences. Positioned within the broader [[blueally-presentation-toolkit]] alongside [[blueally-workshop-experience]] and [[blueally-presenting]], representing BlueAlly's broader shift from PowerPoint to web-native presentation formats.
