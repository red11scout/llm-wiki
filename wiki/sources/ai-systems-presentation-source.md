## Description

[[ai-systems-presentation]] is an interactive animated web presentation on AI systems architecture. It is a web-based replacement for traditional PowerPoint slides, using [[framer-motion]] for slide transitions and animations, with [[zustand]] managing presentation state and navigation. No database layer is required.
