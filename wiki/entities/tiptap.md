Tiptap is a headless, extensible rich text editor framework built on top of ProseMirror. It is framework-agnostic and widely used in React and Next.js applications for structured content authoring.

## Usage in BlueAlly

Tiptap is integrated into [[blueally-presenting]] as the content-authoring layer, enabling editors to create and manage presentation nodes within the [[knowledge-graph-presentation]] data model alongside an admin dashboard.
