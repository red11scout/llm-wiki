Four Cups is a personal/religious Next.js application deployed at fourcups.vercel.app. It is built by the same author as the BlueAlly ecosystem but is explicitly not part of it. The app uses [[neon]] for persistence, [[drizzle-orm]] for data access, and [[framer-motion]] for animations.

See [[fourcups-source]] for the raw project inventory.

## Stack

- **Framework**: Next.js (built on [[react]])
- **Database**: [[neon]] (serverless Postgres)
- **ORM**: [[drizzle-orm]]
- **Animations**: [[framer-motion]]
- **Deployment**: [[vercel]]

## Ecosystem role

Personal project. Shares the Neon + Drizzle ORM stack with BlueAlly production apps and with the other personal project [[godwin-family-tracker]], illustrating the author's consistent technology baseline across personal and professional work.
