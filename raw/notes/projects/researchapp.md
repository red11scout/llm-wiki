## Description

BlueAlly AI Strategic Assessment Platform, the original Express/Vite version. Provides AI-powered strategic assessment capabilities including use case discovery, financial modeling, and multi-agent analysis. Features enriched use case fields that are all optional for backward compatibility, retry loops with read-back verification for save reliability, and Framer Motion animations.

## Ecosystem role

Entry point to the BlueAlly ecosystem at discover.movefasterwithai.com (step 1). Runs AI assessments and exports JSON that flows into aiworkflow for analysis, then into ai-architecture-studio for technical artifacts. The original assessment tool that started the entire pipeline concept.
