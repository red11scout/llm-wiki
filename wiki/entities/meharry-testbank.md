Meharry Test Bank is a deployed web application built for [[meharry-medical-college]] as a standalone external-client project. It provides a structured repository of test questions and assessment materials, with database persistence for content management and student interaction tracking.

## Technical stack

- **Framework**: [[nextjs]] with [[react]]
- **Styling**: [[tailwindcss]]
- **ORM**: [[drizzle-orm]]
- **Database**: [[neon]] (serverless Postgres)
- **Hosting**: [[vercel]] at meharry-testbank.vercel.app
- **Last commit**: 2026-04-07

## Ecosystem role

This is a standalone external-client delivery for the healthcare education sector. It is not part of the [[blueally-pipeline]] or any BlueAlly AI assessment tooling. It shares infrastructure patterns (Next.js, Neon, Drizzle ORM, Vercel) common across the author's project portfolio but serves an independent client domain.
