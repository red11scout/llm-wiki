AI Systems Presentation is a deployed BlueAlly web application that delivers an interactive, animated presentation on AI systems architecture. It is a web-native replacement for traditional PowerPoint slides, using [[framer-motion]] for slide transitions and animations and [[zustand]] for presentation state and navigation management. It has no database layer.

## Technical details

- **Stack**: Next.js, React, [[tailwindcss]], [[framer-motion]], [[zustand]]
- **Database**: none
- **Deployed**: ai-systems-presentation.vercel.app (via [[vercel]])
- **GitHub**: https://github.com/red11scout/ai-systems-presentation
- **Last commit**: 2026-03-26

## Ecosystem role

Part of the [[blueally-presentation-toolkit]], alongside [[ai-executive-briefing]], [[blueally-workshop-experience]], and [[blueally-presenting]]. Functions as a reusable template for web-native technical presentations on AI systems architecture, reinforcing BlueAlly's shift away from static slide decks.
