An MCP (Model Context Protocol) server is a local or remote process that extends a [[claude-code]] environment with additional capabilities — such as tools, resources, or persistent memory. The BlueAlly development environment is configured with 9 MCP servers; [[claude-mem]] is one of them, providing memory compression and session persistence.
