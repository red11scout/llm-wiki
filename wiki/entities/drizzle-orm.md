Drizzle ORM is a TypeScript-native ORM library used within the BlueAlly stack for type-safe database access against Postgres. It appears in multiple BlueAlly applications as the preferred ORM layer.

## Usage in BlueAlly

- [[ai-catalyst]] — used for database access in the archived workshop analysis platform.
- [[ai-executive-readout]] — used for database-backed content retrieval in the executive readout report generator.
