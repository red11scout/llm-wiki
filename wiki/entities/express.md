Express is a minimal Node.js web framework used as the back-end server layer across BlueAlly tools and personal projects by the same author. It provides routing and HTTP handling in applications where a lightweight server layer is preferred over a full framework.

## Used in

### BlueAlly ecosystem
- [[claude-mem]] — MCP server HTTP layer
- [[blueally-intelligence]] — archived M&A analytics platform
- [[cognoresearcher]] — archived strategic assessment platform
- Early-generation React + Vite + Express BlueAlly stack

### Personal projects
- [[godwin-family-tracker]] — personal family task-scoring app; also uses Express for WebSocket handling
