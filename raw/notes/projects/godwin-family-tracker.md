## Description

Daily task-scoring app for the 6 Godwin children (Scout, Pace, Arch, Bryce, Knox, Wylder) designed for display on the living room TV. Features 6 scoring categories across 3 dimensions (completion, attitude, timeliness) on a 1-5 scale for a max of 90 points per day. All math is deterministic in shared/scoring.ts with AI used only for generating prose encouragement. Includes WebSocket live updates, PIN authentication, and Gold/Silver/Bronze tier rankings.

## Ecosystem role

Personal family project, not part of the BlueAlly ecosystem. Built using the same React + Vite + Express + Drizzle + Neon stack as early-generation BlueAlly tools.
