Zustand is a lightweight, hook-based React state management library. In the BlueAlly ecosystem it is the standard choice for client-side state where a full server-state solution is not warranted.

## Usage in BlueAlly apps

- [[blueally-workshop-experience]] — manages presentation state (current slide, navigation) for the 28-slide live workshop platform
- Multiple other BlueAlly apps use Zustand for local UI and workflow state
