Next.js is a full-stack React web framework developed and maintained by [[vercel]]. It provides server-side rendering, static site generation, API routes, and file-based routing, making it the dominant application framework across BlueAlly's deployed product portfolio.

## Usage across BlueAlly ecosystem

Next.js is used in the majority of BlueAlly apps and external-client tools, including but not limited to: [[aiworkflow]], [[blueally-portal]], [[blueally-microsites]], [[ai-visual-intelligence-library]], [[blueally-workshop-web]], [[blueally-private-ai-investment-planner]], [[blueally-ai-app]], [[blueally-ai-solution-navigator]], [[aiwebsiteblueally-next]], [[mativ-roi-dashboard]], and [[ga-staff-scheduler]].
